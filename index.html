We have used curl c++ library to be able to send commands while the robot server answers only to http commands.

functions :

captureImage();

Calling This function will send a image capture command to server in order to take a picture with the camera connect to endpoint of arm.

The taken picture will be saved in this link http://debatedecide.fit.edu/robot/last.bmp

saveImage(string path_to_images);

This function downloads the latest image taken by captureImage(); and saves it to the path given by user as a parameter to function.The message will be returned based on if the execution was successful or failed with error message .

sendCommand_TMOVETO(int hand,int wrist,int x,int y,int z);

Calling this function will send a TMOVETO command to arm with the parameters hand as hand position , wrist as wrist position and x,y,z parameters for the robot to move endpoint to desired x,y,z position.

sendCommand_HOME();

This Function will send a command to arm to go its home position.

captureSave(std::string path);

This Function is combination of function capture image and save image with will constantly take picture and save it to the path defined by user as an input parameter to function

Algorithm : There are  2 main loops one for scanning front and one for top side . given correct values as position of object in space the algorithm will start to scan the front side of object from one corner to another corner by taking small steps and while each step it will take a picture and save it to the local machine (this will be used later to process images to find objects).

The value of password variable needs to be set so the program can work otherwise the curl lib will return error message.

How to run code :

 

This code needs curl lib to compile :

Install curl in ubuntu:

    sudo apt-get install curl

Install curl in Mac :

    sudo apt-get install curl

or

    brew install curl

Compile :

    cd R12-Arm

    make

    ./arm
