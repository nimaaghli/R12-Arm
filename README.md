# R12-Arm
R12 Robotic Arm C++ interface

#This code needs curl lib to compile :

##Install curl in ubuntu:

  >sudo apt-get install curl
  
##Install curl in Mac :

  >sudo apt-get install curl
  
  or
  
  >brew install curl
##OpenCV 2.4.11 is needed to compile branch project 2  
  
#Compile : 
  >cd R12-Arm 
  
  >make
  
  >./arm <password> <path to save images>
